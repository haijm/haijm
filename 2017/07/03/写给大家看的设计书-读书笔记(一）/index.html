写给大家看的设计书-读书笔记（一）

Share

写给大家看的设计书

一、亲密性

  • Robin亲密性原则
    • 将相关的项组织在一起，移动这些项，是他们的物理位置相互靠近，这样一来，相关的项将被看作凝聚为一体的一个组，而不再是一对彼此无关的片段。
    • 将类似的项归为一组来建立很近的亲密性是，有时需要做一些修改，如需调整文本的大小或字体粗细，或者要改变图片的大小或放置位置，等等。
    • 亲密性的思想并不是说所有一切都要更靠近，其真正的含义是：如果某些元素在礼节性上存在关联，或者相互之间存在某种关系，那么这些元素在视觉上也应当有关联。除此之外，其他孤立的元素火元素组则不应存在亲密性。位置是否靠近可以体现出元素之间是否存在关系。
    • 将有关系的项归为一组。如果页面上有些区域的组织不太清晰，则要看看这些想是不是存在本不该有的亲密性。使用这个简单的设计原则来管理空间，不仅会使页面更有条理，看上去也更美观。

      亲密性小结

  • 如果多个项相互之间存在很近的亲密性，他们将成为以视觉单元而不是多个孤立的元素。彼此相关的项应当归在一组。要有意识的注意你时怎样阅读的，你的实现怎样移动：从哪里开始；沿着怎样的路径；到哪里结束；读完之后，接下来看哪里？整个过程应当是一个合理的过程，有确定的开始，而且要有确定的结束。
  • 根本目的
    亲密性的根本目的是实现组织性。尽管其他原则也能达到这个目的。不过利用亲密性原则，只需要简单的将相关的元素分在一组简历更近的亲密性，就能自动的实现条理性和组织性。如果信息很有条理，将更容易阅读，也更容易被记住。此外还有一个很好的“副产品”，利用亲密性原则，还可以是空白更美观也更有条理。
  • 如果实现
    微微眯起眼，统计你的眼睛停顿的次数来数一数页面上有多少个元素。如果页面上的项超过3~5个，就要看看哪些孤立的元素可以跪在一组简历更近的亲密性，使之成为一个视觉单位。
  • 要避免的问题
    不要仅仅因为有空白就把元素放在角落或中央。避免一个页面上有太多孤立的元素。不要再元素之间留出同样大小的空白，除非各组同属于一个子集。标题、子标题、图表标题、图片能否归入其相关材料？在这个问题上一定要非常清楚。再有很近亲密性的元素之间建立关系。不同属一组的元素之间不要建立关系。如果元素彼此无关，要把它们分开。

    二、对齐

  • Robin对齐原则
    • 任何元素都不能在页面上随意安放。每一项都应该于页面上的某个内容存在某种视觉联系。对齐原则要求特别小心，在不能像从前那样，只要页面上刚好有空间就把元素随意“扔”在那里。
    • 在设计中，统一性是一个重要的概念。要让页面上的所有元素看上去同意、有联系切彼此相关，需要在哥哥单独的元素之间存在某种视觉纽带。尽管这些鼓励元素在页面上的物理位置可能并不靠近，但是通过适当放置，可以让他们看上去是由联系而且相关的。并且与其他信息统一。可以看看你喜欢的那些设计。一个精美的设计不论最初看上去多么杂乱无章，总能找出其中的对齐方式。
  • 根本目的
    对其的根本目的是是页面统一而且有条理。其效果类似于把客厅里四处零落的洋娃娃捡起来，并把它们放在一个玩具箱中。不论创建精美的，正式的，有趣的还是严肃的外观，通常都可以利用一种明确的对齐（当然，要结合适当的字体）来达到目的。
  • 如何实现
    要特别注意元素放在哪里。应当总能在页面上找出与之对其的元素，尽管这两个对象的物理位置可能相距很远。
  • 要避免的问题
    要避免在页面上混合使用多种文本对齐方式（也就是说，不要将某些文本居中，而另外一些文本右对齐）。另外，要着力避免居中对齐，除非你有意识的想要创建一种比较正式、稳重（通常也更乏味）的表示。并不是完全杜绝使用居中对齐，有时可以有意的选择这种对齐方式，但是不要不把他作为默认选择。

    三、重复

  • Robin重复原则指出，“设计的某些方面需要在整个作品中重复”。重复元素可能是一种粗体字体、一条粗线，某个项目符号、颜色、设计要素，某种格式、空间关系等。
  • 重复小结
    设计中视觉元素的重复可以将作品中的各个部分连在一起，从而统一并增强整个作品，否则这些部分只是彼此鼓励的单元。重复不仅对只有一夜的作品很有用，对于多页文档的设计更显重要（对此我们通常称之为保持一致）。
  • 根本目的
    重复的目的就是统一，并增强视觉效果。不要低估页面视觉效果的威力，如果一个作品看起来很有趣，它往往也更易于阅读。
  • 如何实现
    重复可以认为是保持一致性，而且我相信你早已经这样做过。现在，需要把现有的一致性更向前推一步。可不可以专门做一个图片设计，并且将某些一致元素（如标题）作为这个图片设计的一部分？在每个页面底部或每个标题下面使用一条粗细为1点的线吗？或者是不是使用一条更粗的线（4点），使重复元素显得更明显，更生动？
  • 要避免的问题
    要避免太多的重复一个元素，重复太多会让人讨厌，要注意对比的价值。

    四、对比

  • Robin对比原则
    如果两个项不完全相同，就应当使之不同，而且应当是截然不同。可以采用多种方式产生对比。如字体的大小，粗细的对比，细线与粗线，冷色与暖色，平滑材质与粗糙材质的对比，水平元素与垂直元素，间隔很宽的文本与紧凑在一起的文本形成对比，大图片于小图片，等等。
  • 对比小结
    在页面上增加对比能吸引人的眼球。在我们的眼睛喜欢看到对比的事物。如果页面上放两个不完全相同的元素（比如有两种不同字体，或者两种不同线宽），他们就不能类似。要现实有效的对比，这两个元素必须截然不同。
  • 根本目的
    可以通过字体选择、线宽、颜色、形状、大小、空间等来增加对比，增加对比很容易，途径有很多，这可能是增加视觉效果最有意思也最让人满意的方法了。重要的是：对比一定要强烈。
  • 要避免的问题
    不要犹豫。如果你想形成对比，就加大力度。不要将一种粗线与一种更粗的线进行对比。不要将棕色文本与黑色标题建立对比。要避免使用两种或多种类似的字体。如果各个项不完全一样，干脆让他们截然不同。
